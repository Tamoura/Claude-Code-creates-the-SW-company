## Deployment Steps Overview

The deployment follows this sequence:

### Phase 1: Database Setup (5 minutes)
1. Create PostgreSQL database on Render
2. Copy connection string
3. Verify database is active

### Phase 2: Backend API (10 minutes)
1. Create web service on Render
2. Connect GitHub repository
3. Configure build commands
4. Set environment variables
5. Deploy and verify health endpoint

### Phase 3: Frontend Web App (10 minutes)
1. Create project on Vercel
2. Connect GitHub repository
3. Configure build settings
4. Set API URL environment variable
5. Deploy and verify login page

### Phase 4: Security & Testing (5 minutes)
1. Update CORS to restrict to frontend URL
2. Create admin user
3. Test full application flow

**Total Time**: 30 minutes
