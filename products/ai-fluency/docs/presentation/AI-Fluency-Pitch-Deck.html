Scoring Algorithm

Prevalence-weighted scoring derived from Anthropic's 9,830-conversation dataset. Each behavioral indicator has empirical prevalence rates that weight its contribution to dimension scores.

Key innovation: Discernment gap detection identifies where users think they're proficient but behavioral data says otherwise.

Security & Compliance

  • Multi-tenant isolation via PostgreSQL Row-Level Security — data physically separated per tenant
  • GDPR/CCPA compliant — configurable data retention, right to deletion, consent management
  • SOC 2 Type II target (Year 2)
  • Encryption — TLS 1.3 in transit, AES-256 at rest
  • Audit logging — every assessment action logged for compliance
  • RBAC — Platform Admin, Org Admin, Manager, Learner roles

Integration Architecture

  • LTI 1.3 — Canvas, Moodle, Blackboard (grade passback)
